⁷# PhotoStore — Modern Web Application

**PhotoStore** — это динамическое веб-приложение для поиска и управления визуальным контентом. Проект построен с использованием современного стека Frontend-разработки, фокусируясь на производительности и удобстве интерфейса.

## 🛠 Технологический стек и функционал

В проекте реализованы следующие ключевые технологии:

1.  **Vite**: Использован как инструмент сборки для обеспечения мгновенного HMR (Hot Module Replacement) и оптимизированной сборки проекта.
2.  **JS Modules**: Код разделен на независимые модули (API, UI-render, Handlers), что улучшает читаемость и поддержку проекта.
3.  **DOM Manipulation**: Динамическое создание элементов галереи, модальных окон и фильтров без перезагрузки страницы.
4.  **Event Handling**: Обработка сложных пользовательских сценариев: поиск при вводе (input), клики по категориям, управление модальными окнами и адаптивное меню.
5.  **Form Validation**: Реализована клиентская валидация форм (например, форма подписки или контактов) для предотвращения отправки некорректных данных.
6.  **Storage (Local/Session)**: Сохранение состояния пользователя (например, «Избранное» или настройки темы) локально в браузере.
7.  **Async/Await & Fetch API**: Интеграция с внешним API для получения данных в реальном времени (фотографии, погода или новости) с обработкой состояний загрузки и ошибок.
